구름이 데려오기 ☁️

순수 CSS 도형으로 그린 오리지널 캐릭터라 저작권/카피 이슈가 없어요. 아래 버튼을 누르면 설치 없이 이 페이지에서 바로 구름이를 만날 수 있습니다.

모든 사이트 위자유 배회드래그 이동 ⭐ 별 먹이주기💬 말걸기😴 낮잠
💡 크롬에 정식 설치 (모든 사이트 위에서 돌아다니게):
주소창에 chrome://extensions → 우측 상단 개발자 모드 켜기 → 압축해제된 확장 프로그램을 로드 → deskmate-extension 폴더 선택.
설치 후 아무 웹사이트에서나 구름이가 등장합니다. 확장 아이콘을 누르면 켜기/끄기.
