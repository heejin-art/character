# ☁️ 구름이 · 오버레이 데스크 펫

어떤 웹사이트를 띄워놓고 작업해도 **그 화면 위를 둥실둥실 떠다니는** 뭉게구름 펫.
걷고, 낮잠 자고, 클릭하면 반응하고, 별을 주면 먹으러 쪼르르 달려갑니다.

> 크롬 확장(Manifest V3)으로 설치하면 **모든 사이트 위**에서 구름이가 등장해요.

## ✨ 컨셉

기존 IP(포켓몬·산리오·라인/카카오프렌즈·다마고치 등)와 전혀 겹치지 않도록,
**순수 CSS 도형으로 그린 오리지널 뭉게구름 "구름이"** 로 디자인했습니다.
눈·볼터치·입만으로 표정을 만드는 스타일이라 카피 이슈가 없고, 외부 에셋/이미지도 쓰지 않습니다.

## 🕹️ 기능

- **자유 배회** — 화면 바닥을 아장아장 걸어다니고, 가끔 낮잠(😴 Zzz)을 잠
- **클릭 반응** — 통통 튀기 + 말풍선 + 하트 파티클
- **드래그 이동** — 마우스로 집어 원하는 위치에 놓으면 사뿐히 착지 (터치 지원)
- **⭐ 별 먹이주기** — 화면에 별이 나타나면 구름이가 **먹으러 걸어감**
- **💬 말걸기** — 응원·혼잣말 대사
- **켜기/끄기** — 확장 아이콘 클릭으로 현재 탭에서 토글

## ▶️ 설치 & 실행

### 크롬 확장으로 설치 (권장 — 모든 사이트 위에서 돌아다님)

1. 크롬 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램을 로드** → `deskmate-extension` 폴더 선택
4. 아무 웹사이트에서나 구름이 등장 · 확장 아이콘으로 켜기/끄기

### 설치 없이 미리보기

의존성/빌드 없음. 로컬 서버로 랜딩 페이지를 열고 **"이 페이지에서 바로 만나기 ▶"** 클릭:

```bash
python -m http.server 8000      # 또는 아무 정적 서버
# → http://localhost:8000
```

## 📁 구조

```
index.html                      # 랜딩 — 구름이 소개 + 미리보기 + 설치 안내
deskmate-extension/             # ★ 메인: 오버레이 펫 크롬 확장 (MV3)
  manifest.json                 #   확장 설정
  content.js                    #   구름이 로직 (배회·드래그·먹이·대화·낮잠)
  pet.css                       #   구름이 캐릭터 디자인 · 애니메이션
  background.js                 #   아이콘 클릭 → 켜기/끄기 토글
buddies/                        # (실험작) 말랑 친구들 웹앱
pixel/                          # (실험작) 픽셀 다마고치 또리 웹앱
```

## 🗺️ 향후 로드맵 (아이디어)

- 여러 마리 동시 등장 / 캐릭터 스킨(구름이 외 모찌·콩이·별똥이)
- 창 가장자리 걷기 · 창 위로 올라타기, 마우스 따라오기
- 감정 상태 확장(졸림/신남/삐짐), 시간대별 행동, 먹이 종류
- Electron 래핑 → 진짜 데스크탑 위를 떠다니는 투명 창 펫
- 수익화 여지: 캐릭터 스킨/코스튬, 시즌 한정 테마

## 🛠️ 기술 스택

순수 HTML/CSS/JS (프레임워크 없음). 캐릭터는 전부 CSS 도형으로 그려 외부 에셋/저작권 문제 없음.
