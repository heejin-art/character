# 🐣 말랑 친구들 · Floating Buddies

떠다니고 돌아다니는 아주 귀여운 데스크탑/모바일 펫. 클릭하면 반응하고, 먹이를 주고, 말을 거는 힐링 위젯.

## ✨ 컨셉

기존 IP(포켓몬·산리오·라인/카카오프렌즈·다마고치 등)와 전혀 겹치지 않도록,
**음식·자연·천체 모티프의 미니멀 "말랑 블롭(blob)"** 으로 오리지널 디자인했습니다.
눈 · 볼터치 · 입만으로 표정을 만드는 스타일이라 카피 이슈가 없고, 누구나 귀엽다고 느낄 보편적 매력을 노렸습니다.

## 🎭 오리지널 캐릭터 4종

| 캐릭터 | 모티프 | 성격 | 대표 대사 |
|---|---|---|---|
| **모찌** 🍡 | 말랑 떡/모찌 | 졸린 힐링형 | "말랑말랑~", "쓰담쓰담 해줘잉" |
| **방울** 💧 | 물방울 요정 | 발랄 상쾌형 | "퐁당!", "오늘 기분 상쾌해~" |
| **콩이** 🌱 | 새싹 콩 | 수줍은 성실형 | "쑥쑥 자랄래", "물 주세용" |
| **별똥이** ✨ | 작은 별똥별 | 몽환 장난꾸러기 | "반짝반짝", "소원 빌어줄게" |

## 🕹️ 기능

- **자유 배회** — 화면 위를 둥둥 떠다니며 돌아다님
- **클릭 반응** — 통통 튀기 + 말풍선 + 하트 파티클
- **드래그 이동** — 원하는 위치로 옮기기 (모바일 터치 지원)
- **먹이주기 / 쓰다듬기 / 말걸기** — 상호작용 버튼
- **스탯 시스템** — 배고픔·기분이 시간에 따라 감소, 돌봐주면 회복
- **캐릭터 교체** — 마음에 드는 친구 선택 (배경 무드도 함께 변화)

## ▶️ 로컬에서 실행

의존성/빌드 없음. 두 가지 방법:

```bash
# 1) 그냥 파일 열기
#    index.html 더블클릭

# 2) 로컬 서버 (권장)
python -m http.server 8000
# → http://localhost:8000
```

## 🗺️ 향후 로드맵 (아이디어)

- **v1 (현재):** 웹 데모 — 배회 · 반응 · 먹이 · 대화 · 4캐릭터
- **v2:** Electron 래핑 → 진짜 "데스크탑 위를 떠다니는" 투명 창 펫
- **v2:** 모바일 PWA / 위젯화, 화면 가장자리 걷기 · 창 위로 올라타기
- **v3:** 감정 상태 확장(졸림/신남/삐짐), 시간대별 행동, 미니 인벤토리(먹이 종류), 성장 단계
- **수익화 여지:** 캐릭터 스킨/코스튬, 시즌 한정 테마, 이모티콘 팩

## 🛠️ 기술 스택

순수 HTML/CSS/JS (프레임워크 없음). 캐릭터는 전부 CSS 도형으로 그려 외부 에셋/저작권 문제 없음.

## 📁 구조

```
index.html   # 구조 + UI
styles.css   # 캐릭터 디자인 · 애니메이션 · 테마
app.js       # 배회 · 상호작용 · 스탯 로직
```
